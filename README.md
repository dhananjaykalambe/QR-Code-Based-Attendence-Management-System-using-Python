# QR-Code-Based-Attendence-Management-System-using-Python
##  Description

This project is used to automate attendance using QR codes. It helps reduce manual work and improves accuracy.

##  Features

* Fast attendance marking
* Reduces manual errors
* Easy to use system

##  Technologies Used

* Python
* Arduino

##  How to Run

1. Install Python
2. Run the main file
3. Scan QR code
