About Project

Project Name: Smart Attendance System

Technologies Used:

  • Flask (Python): Used to develop the backend and handle routing, requests, and server logic.
  • HTML: Used to design the structure of web pages.
  • CSS: Used for styling and improving the user interface.
  • JavaScript: Used for client-side functionality like QR scanning and dynamic behavior.
  • SQLite: Used as a lightweight database to store student and attendance records.
  • Chart.js: Used to display graphical representation of attendance data.
  • HTML5 QR Code Scanner: Used to scan QR codes directly from the browser.
  • qrcode Python Library: Used to generate QR codes for attendance sessions.

Features:

  • QR Code Based Attendance
  • Session Expiry System
  • Duplicate Entry Prevention
  • Student Database Management
